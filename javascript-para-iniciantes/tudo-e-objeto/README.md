# Tudo é objeto

## Tudo é objeto

Strings, Números, Boolean, Objetos e mais, possuem propriedades
e métodos. Por isso são objetos.

```js
    var nome = 'André';
    nome.length; // 5
    nome.charAt(1); // 'n'
    nome.replace('ré', 'rei'); // 'Andrei'
    nome; // 'André'
```

*Uma string herda propriedades e*
*métodos do construtor String()*
