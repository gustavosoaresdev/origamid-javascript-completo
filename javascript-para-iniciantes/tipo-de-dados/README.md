# Tipos de Dados

## 7 Tipos de Dados

Todos são primitivos exceto os objetos.

```js
    var nome = 'Gustavo';    //  String
    var idade = 19;    //  Number
    var possuiFaculdade = false;    //  Boolean
    var time;    //  Undefined
    var comida = null;    //  Null
    var simbolo = Symbol();    //  Symbol
    var novoObjeto = {};    //  Object
```

*Primitivos são dados imutáveis.*
