# Arrays e Loops

## Array

É um grupo de valores geralmente relacionados. Servem para
guardarmos diferentes valores em uma única variável.

```js
    var videoGames = ['Switch', 'PS4', 'XBox'];

    videoGames[0]; // Switch
    videoGames[2]; // Xbox
```
